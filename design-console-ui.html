CONSOLE UI CONCEPT

Tchiki-Tchiki reimagined as a terminal. Every screen = console output. Buttons = commands.

01 — Boot / Home
The app opens like a terminal booting up. Name entry is a command prompt. Create/Join are executable commands.
tchiki-tchiki v1.0
GNU Tchiki-Tchiki 1.0.0
Copyright (c) 2025 tchiki-tchiki
Type 'help' for available commands.
